PDF Search Test Page

This page helps test the PDF search functionality of your extension.

Instructions:

  1. Open a PDF file in your browser (preferably a local PDF file)
  2. Open the browser's Developer Console (F12)
  3. Type debugPdfSearch() in the console to see debug information
  4. Try searching for words using your extension (Ctrl+F or click the floating search button)
  5. Check the console for any error messages or debug output

Debugging Commands:

In the browser console, you can run these commands:

  • debugPdfSearch() - Shows PDF structure and content information
  • console.log('PDF detected:', isPDFPage()) - Check if PDF is detected
  • console.log('Text content length:', pdfTextContent.length) - Check extracted text length

Common Issues and Solutions:

  • Text not found in PDF: The PDF text might be in images (scanned PDF) or use different text layer structure
  • Search not working: Try waiting a few seconds after opening the PDF for text extraction to complete
  • No floating button: Make sure the extension is enabled and has permission for file URLs
